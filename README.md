# Simple express CRUD app

this is a task in the Zuri internship.

it is meant to creat,read,update and delete data from mongodb

## Author

- [@EbukaNathan](https://www.github.com/ebukanathan)
 

## API Reference

#### creat item
POST  /info
#### fetch item by id

GET /info/${id}

### fetch all item
GET  /info

### Update item

PUT /info/${id}


###Delete

Delete /info/${id}
